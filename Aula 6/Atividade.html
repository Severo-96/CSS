METODOLOGIA

Metodologia de Pesquisa

Esse estudo foi desenvolvido sendo adotado um método de pesquisa misto, utilizando tanto pesquisa documental quanto análise de dados públicos fornecidos por órgãos federais, como a Agência Nacional de Aviação Civil (ANAC), além de outras fontes secundárias.

Para definir os anos de estudo foram considerados os dados de todos os anos disponibilizados pela ANAC, visto que, na estatística quanto maior a amostra dos dados menor será o erro apresentado nos estudos. Diante disso, o período de estudo adotado para modelagem foram os anos entre 2000 e 2019 para modelagem, além disso, os anos 2020 e 2021 foram utilizados para fazer a análise comparativa dos efeitos da pandemia sobre a variação de demanda. Este período de estudo permite a visualização do comportamento setor aéreo internacional antes e durante a pandemia.

Montagem da base de dados

A base de dados foi feita com as informações obtidas através Consulta Interativa dos Indicadores do Mercado de Transporte Aéreo (Demanda e Oferta), fornecidos pela ANAC. Sendo esses dados: A quantidade mensal de voos internacionais e a quantidade mensal de passageiros pagos transportados em voos internacionais dos anos de 2000 ao de 2021.

As informações foram organizadas com o auxílio do software EXCEL, sendo essa uma ferramenta do pacote OFFICE, criado e administrado pela empresa MICROSOFT CORPORATION. Extremamente eficaz para armazenamento, organização e tratamento de dados estatísticos, por conta de suas planilhas, funções e fórmulas.

Os dados foram organizados de duas formas, uma delas sendo a quantidade anual e a quantidade trimestral (número de voos e número de passageiros), para que se possa melhor analisar a variação deles e os impactos da pandemia nos números.

Depois de organizados esses dados, foi montada a base de dados da modelagem com os anos de 2000 até 2019, com eles foram criados gráficos com o intuito de encontrar a melhor linha de tendência que representasse a variação de valores.

Com os gráficos em mãos, foi utilizado a função do EXCEL: “Adicionar Linha de Tendência”, que fornece diversas equações que representam a linha de tendência do gráfico, além de mostrar o coeficiente de determinação (R²) de cada uma.
